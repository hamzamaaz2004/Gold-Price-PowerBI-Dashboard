# Gold Price Power BI Dashboard

##  Project Overview
This project is a **Power BI Dashboard** developed to analyze and visualize **Gold Price trends** using interactive reports and charts. It helps users understand price movements over time and gain valuable insights through data visualization.

##  Dashboard Features
- Interactive dashboard design  
- Gold price trend analysis  
- Historical price comparison  
- Charts and graphs for better understanding  
- Filters and slicers for customized insights  
- Clean and user-friendly layout  

##  Tools & Technologies Used
- **Power BI**
- Data Analysis
- Data Visualization

## File Included
- `GOlD Price.pbix` – Main Power BI dashboard file

##  How to Use
1. Download the repository files.
2. Open `GOlD Price.pbix` in **Microsoft Power BI Desktop**.
3. Explore the dashboard using filters and slicers.
4. Analyze gold price trends and insights.

##  Project Purpose
This project is created for:
- Students learning Power BI  
- Beginners practicing dashboard creation  
- Data analysts exploring visualization techniques  
- Anyone interested in gold price trend analysis  

##  Insights You Can Get
- Gold price changes over time  
- Highest and lowest price periods  
- Trend patterns and comparisons  
- Interactive data exploration
